  - ```git branch name``` name이라는 이름의 브랜치 생성
  - ```git checkout name``` 현재 git head를 name이라는 브랜치로 이동
  - 이 이후에는 사실 IDE에서 commit, push 하는게 더 편한 것 같습니다.
